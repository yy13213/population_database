# 🧪 快速测试指南

## 启动应用

```bash
cd GIS_Flask
python app.py
```

## 测试步骤

### 1. 访问主页

打开浏览器访问：
```
http://127.0.0.1:6667
```

### 2. 打开开发者工具

按 `F12` 或 `Ctrl+Shift+I`

### 3. 查看控制台

**成功的输出应该是：**

```
📥 正在加载中国地图数据...
✅ 中国地图数据加载成功 (阿里云DataV)
   - 省份数量: 34
✅ 数据加载成功
   - 总人口: 132,546
   - 省份数: 34
```

**如果看到错误：**

```
❌ 主数据源加载失败: [错误信息]
🔄 尝试备用地图数据源 (GitHub)...
✅ 备用地图数据加载成功 (GitHub)
```

这是正常的，备用数据源会自动启用。

### 4. 验证地图显示

- 点击"人口分布"标签
- 应该能看到完整的中国地图
- 鼠标悬停在省份上应该显示数据

### 5. 测试所有标签

依次点击测试：
- ✅ 人口分布
- ✅ 人口密度  
- ✅ 婚姻统计
- ✅ 人口迁移

## 常见问题

### 问题1: 地图不显示

**解决方案**：
1. 刷新页面（Ctrl+F5）
2. 清除浏览器缓存
3. 检查网络连接

### 问题2: 控制台显示CORS错误

**解决方案**：
- 这是正常的，应用会自动切换到备用数据源

### 问题3: 数据加载很慢

**解决方案**：
- 第一次加载需要从远程获取数据
- 后续加载会使用缓存
