# 🎨 蓝色科幻风格 UI 设计说明

## 设计理念

### 🎯 设计目标
- **科幻感**：深色背景 + 蓝色高光
- **扁平化**：去除阴影和渐变，简洁线条
- **专业性**：适合数据可视化场景
- **易读性**：高对比度文字

---

## 🧩 UI组件

### 1. 头部导航

```css
背景: #1a1f3a
标题颜色: #00d4ff
副标题: #8a92a6
边框: 1px solid #2d3548
```

**设计特点**：
- ✅ 简洁的单色背景
- ✅ 科技蓝标题突出
- ✅ 细线分隔，不喧宾夺主

### 2. 状态栏

```css
背景: #1a1f3a
文字: #8a92a6
在线指示: #00ff88 (脉冲动画)
按钮: #00d4ff
```

**设计特点**：
- ✅ 荧光绿脉冲表示系统运行
- ✅ 扁平按钮，无阴影
- ✅ 悬停时轻微颜色变化

### 3. 统计卡片

```css
背景: #1a1f3a
边框: 1px solid #2d3548
数值颜色: #00d4ff
标签颜色: #8a92a6
悬停边框: #00d4ff
```

**设计特点**：
- ✅ 方形卡片，无圆角
- ✅ 细边框，扁平化
- ✅ 悬停时边框亮蓝色
- ✅ 无阴影效果

**示例**：
```
┌────────────────────┐
│  📊                │
│  132,546          │ ← #00d4ff
│  总人口数          │ ← #8a92a6
└────────────────────┘
  #2d3548 边框
```

### 4. 导航标签

```css
背景: 透明
激活背景: rgba(0, 212, 255, 0.08)
文字: #8a92a6
激活文字: #00d4ff
激活边框: 2px底部蓝线
```

**设计特点**：
- ✅ 扁平标签，无背景
- ✅ 激活时底部蓝线
- ✅ 轻微半透明蓝色背景
- ✅ 无圆角，无阴影

**示例**：
```
┌─────────┬─────────┬─────────┐
│ 人口分布 │ 人口密度 │ 婚姻统计 │
└─────────┴─────────┴─────────┘
    ━━━━━     ← 激活状态（蓝线）
```

### 5. 图表容器

```css
背景: #1a1f3a
边框: 1px solid #2d3548
内边距: 24px
```

**设计特点**：
- ✅ 简单矩形容器
- ✅ 细边框分隔
- ✅ 无圆角，无阴影
- ✅ 纯色背景

## 🎭 交互效果

### 悬停效果

```css
/* 卡片悬停 */
border-color: #2d3548 → #00d4ff
background: #1a1f3a → #1e2440

/* 按钮悬停 */
background: #00d4ff → #00b8e6
```

**原则**：
- ✅ 颜色变化，不变大小
- ✅ 无阴影、无缩放
- ✅ 过渡时间 0.3s

### 动画效果

#### 1. 脉冲动画（在线状态）

```css
@keyframes pulse {
    0%, 100% { 
        opacity: 1; 
        box-shadow: 0 0 0 0 rgba(0, 255, 136, 0.7); 
    }
    50% { 
        opacity: 1; 
        box-shadow: 0 0 0 6px rgba(0, 255, 136, 0); 
    }
}
```

**效果**：荧光绿圆点扩散动画

## 🔧 自定义指南

### 更改主色调

如需更改为其他颜色，修改以下变量：

```css
/* 将 #00d4ff (蓝色) 替换为其他颜色 */

/* 例如：改为紫色科技风格 */
#00d4ff → #a855f7  /* 紫色 */

```

### 更改背景亮度

```css
/* 更深色 */
#0a0e27 → #050816  /* 更深的背景 */

/* 更浅色 */
#0a0e27 → #1a1f3a  /* 更浅的背景 */
```
