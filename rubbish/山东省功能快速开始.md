## 📊 数据视图说明

### 视图1: shandong_population

**包含数据**：
- 所有户籍在山东的人口
- 所有现居住在山东的人口

**查询条件**：
```sql
WHERE hukou_province = '山东' OR cur_province = '山东'
```

### 视图2: shandong_deceased

**包含数据**：
- 所有山东省死亡人口

**额外字段**：
- `death_date` - 死亡日期

### 视图3: shandong_marriage

**包含数据**：
- 所有涉及山东省人口的婚姻记录

**查询逻辑**：
- 男方或女方户籍/现居在山东

## 🔌 API使用

### 获取所有数据

```bash
curl http://127.0.0.1:5050/api/shandong/data/all
```

### 获取缓存信息

```bash
curl http://127.0.0.1:5050/api/shandong/cache/info
```

### 强制更新缓存

```bash
curl -X POST http://127.0.0.1:5050/api/shandong/cache/update
```
