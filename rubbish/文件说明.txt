📁 项目文件清单
--------------------------------------------------------------------------------

1. id_card_ocr.py (核心文件)
   - 功能：命令行版本的身份证OCR识别
   - 用法：python id_card_ocr.py <身份证图片路径>
   - 特点：适合批处理和脚本集成

2. id_card_app.py (核心文件)
   - 功能：Streamlit Web界面版本
   - 用法：streamlit run id_card_app.py
   - 特点：图形界面，操作直观

3. requirements.txt (配置文件)
   - 功能：Python依赖包列表
   - 用法：pip install -r requirements.txt
   - 内容：openai, pymysql, streamlit, Pillow

4. README.md (文档)
   - 功能：完整的项目文档
   - 内容：系统简介、安装、使用、技术细节、故障排除

   - 内容：60秒快速开始、使用技巧、常见问题
   - 推荐：首次使用必读

6. 项目总结.md (文档)
   - 功能：项目开发总结
   - 内容：功能列表、技术实现、测试结果、改进方向

   - 内容：所有文件的简要说明

🚀 快速开始
--------------------------------------------------------------------------------

第一次使用？按照以下步骤：

步骤1：安装依赖
  $ cd OCR
  $ pip install -r requirements.txt

步骤2：启动Web界面
  $ streamlit run id_card_app.py

步骤3：使用系统
  1. 在浏览器中打开显示的URL
  2. 上传身份证照片
  3. 点击"开始识别"
  4. 确认信息后保存

就这么简单！

🎯 核心功能
--------------------------------------------------------------------------------

✅ OCR识别        - 使用GPT-4 Vision识别身份证
✅ 自动解析        - 从身份证号解析性别、出生日期
✅ 地址匹配        - 根据地址码匹配省市区
✅ 用户确认        - 显示信息后征得用户同意
✅ 数据入库        - 保存到MySQL数据库

📊 数据字段
--------------------------------------------------------------------------------

从身份证识别的数据包括：

基本信息：
  - 身份证号 (OCR识别)
  - 姓名 (OCR识别)
  - 民族 (OCR识别)

自动解析：
  - 性别 (从身份证号第17位)
  - 出生日期 (从身份证号7-14位)
  - 户籍地址 (从身份证号前6位地址码匹配)

地址解析：
  - 现居住地 (从身份证地址栏解析)

其他：
  - 处理时间 (当前时间)
  - 数据来源 (固定为"image")
