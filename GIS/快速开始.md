# ⚡ 5分钟快速开始

## 📋 前置要求

- ✅ Python 3.8+
- ✅ 数据库可访问
- ✅ `province_data.json` 文件（在上级目录）

---

## 🚀 三步启动

### Step 1: 安装依赖（1分钟）

```bash
cd GIS
pip install -r requirements.txt
```

### Step 2: 测试数据（1分钟）

```bash
python data_statistics.py
```

**预期输出**：
```
============================================================
📊 人口统计数据测试
============================================================

1️⃣ 各省人口数量（前10）:
   1. 广东    :   45,123 人
   ...

✅ 测试完成！
```

### Step 3: 启动Web应用（3分钟）

```bash
streamlit run app.py
```

**自动打开**：`http://localhost:8501`

1. **查看概览**
   - 侧边栏选择：📈 概览统计

## ❓ 遇到问题？

### 问题1：安装失败

```bash
# 使用清华镜像
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 问题2：数据库连接失败

检查 `data_statistics.py` 中的配置：
```python
MYSQL_CONFIG = {

}
```

### 问题3：找不到province_data.json

```bash
# 确保文件在上级目录
ls ../province_data.json

# 或修改代码中的路径

```
